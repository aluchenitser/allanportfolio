about this blog

This is a javascript, front end, and nodejs blog. Its aim is to add to the tech milieu and to elevate my visibiliy as a programmer. You might find it useful!

The physical construction of the blog itself is simple: manually coded and updated html / js / css files hosted on an AWS S3 bucket. If a new feature gets added, I'll write an article describing how I did it.

In general, this blog will be at the level of intermediate and up. If I write about something it probably means that it was something new, intriguing, or a bit challenging to understand. Otherwise it's not too exciting to share.

For beginners, I can recommend codeacademy, w3schools, stackoverflow, and of course Google.

responsive header: use case for css variables

Css variables reduced how much code goes into the responsive header of this blog. Cool eh? This is a typical sticky header that increases from 40px to 70px in desktop sizes:

  • set height for the header
  • add padding to for stickiness
  • center text using line-height



	:root {
		--top-space: 70px;
	}

	.header {
		// ...
		
		margin-top: calc(var(--top-space) * -1);
		line-height: var(--top-space);
		height: var(--top-space);
	}
